# Rock Paper Scissors Game (C++)

## Overview
A simple console-based implementation of the classic Rock-Paper-Scissors game where the user plays against the computer.

## Features
- User vs computer gameplay
- Random move generation for the computer
- Input validation and result display
- Clear win/lose/draw logic

## Technologies
- C++

## How to Run
1. Compile:
g++ rock_paper_scissors.cpp -o game

2. Run:
./game

## Example
Choose (rock/paper/scissors): rock  
Computer: scissors  
Result: You win!

## Learning Outcomes
- Understanding conditional logic
- Working with random number generation
- Building interactive console applications
